## Usage
Run
```sh
sly --profile my-profile --stack-name my-stack
```

Run `sly --help` for more info:
```
Usage:
  sly [flags]

Flags:
  -h, --help                 help for sly
      --profile string       aws profile name
      --stack-name string    aws cfn stack name (required)
```
